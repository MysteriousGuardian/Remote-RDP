# 1. Introduction 

My name is David Tekin, and I’m a bachelor's student in IT-engineering at University of Borås with an interest in networking and cybersecurity. I started this project because I wanted to understand how remote access systems work, similar to the ones we use at school to connect to machines from anywhere.

This is a hands-on project where I’m building my own setup using WireGuard, learning how VPNs function, and exploring networking concepts like routing and connectivity. I’m also experimenting with ways to make the setup more secure over time.

The goal is simply to learn by doing — setting things up, testing what works, breaking things, and improving along the way. By utilizing the equipment I had at home, it gave me a visual on how flexible it is to host a similar project.
